# Language Flashcards App

Веб-приложение для изучения языков с помощью флеш-карточек.

## Функционал
- Добавление карточек
- Режим изучения
- Режим тестирования
- Отслеживание прогресса

## Технологии
- Бэкенд: Python + Flask
- Фронтенд: HTML, CSS, JavaScript
- База данных: SQLite
